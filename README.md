# Simple Biodata Form (PDO Test)

This is a simple test project created to practice **Create** and **Read** operations using PHP PDO.

## About
This is not a full system, but a basic activity to demonstrate:
1.  **Form Fill-up (Create):** A simple HTML form where a user can enter their biodata.
2.  **Data Display (Read):** A page to view the data that was submitted.

## Functionality
* Connects to a MySQL database using PDO.
* Uses a simple form to insert data.
* Fetches and lists the saved records.

## Tech Stack
* PHP (PDO)
* MySQL
* HTML/Bootstrap
